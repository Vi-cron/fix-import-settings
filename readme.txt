Набор фиксов для импорта WordPress: разрешает внешние хосты, увеличивает память, включает отладку.

== Description ==
Плагин решает типичные проблемы при импорте:
* Разрешает скачивание медиафайлов с внешних доменов
* Увеличивает лимит загрузки до 64MB
* Включает режим отладки IMPORT_DEBUG
* Увеличивает лимиты памяти WordPress

== Installation ==
1. Загрузите папку `fix-import-settings` в `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' в WordPress
3. Готово! Плагин работает автоматически
